# Laundry Management System — Frontend

## Overview

Responsive frontend dashboard for managing laundry orders, customer details, order tracking, and status updates.

Built using Vanilla JavaScript, HTML, and CSS.

---

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Vercel Deployment

---

## Features

* Admin Login
* Dashboard Analytics
* Create Orders
* View All Orders
* Search & Filter Orders
* Update Order Status
* Delete Orders
* Print Order Details
* Mobile Responsive UI
* Toast Notifications
* Modal-based UI

---

## Pages

* Login Page
* Dashboard Page
* Orders Page

---

## Frontend Structure

```bash
frontend/
│
├── html/
├── css/
├── js/
└── assets/
```

---

## Deployment

* Frontend Hosted on Vercel
* Connected with Spring Boot Backend APIs
